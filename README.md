# Sky Glass Streaming App

## Description
Sky Glass is a streaming app that enables users to access a variety of content seamlessly. Designed with user experience in mind, it provides a rich interface and robust performance for all types of media.

## Features
- User-friendly interface
- Support for multiple streaming protocols
- Playback controls for a satisfying viewer experience
- Content categorization for easy navigation
- Built-in search functionality

## Setup Instructions
1. Clone the repository:
   ```
   git clone https://github.com/isnoop247-web/web-sky--glass--now.git
   ```
2. Navigate to the project directory:
   ```
   cd web-sky--glass--now
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Local Development Guide
- Ensure you have Node.js installed.
- Use the provided scripts for testing and building the application.
- Follow the existing code style for consistency.

## Deployment Details
1. Build the app for production:
   ```
   npm run build
   ```
2. Deploy the contents of the `build` directory to your preferred hosting service.
